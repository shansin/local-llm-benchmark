Write a Python function called `merge_intervals` that takes a list of intervals (each represented as a tuple of two integers [start, end]) and merges all overlapping intervals. The function should return the merged intervals sorted by start time.

Requirements:
- Handle edge cases: empty list, single interval, already sorted, unsorted input
- Time complexity should be O(n log n)
- Include type hints
- Include 5 test cases that cover edge cases

Example:
  merge_intervals([(1,3), (2,6), (8,10), (15,18)]) => [(1,6), (8,10), (15,18)]
